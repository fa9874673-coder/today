# SIGNAL SCANNER 2.0

一个纯静态的中文互动网页，用柔和的夜间信号星图生成一份轻量、趣味的本地提示。页面不依赖后端或外部 API，不采集、上传或保存个人信息。

## 当前交互

页面包含六颗信号节点：

- 心情频率
- 快乐电量
- 好运浓度
- 休息许可
- 小奖励
- 放空片刻

用户可以依次点亮信号节点。全部点亮后，页面会从本地内容池生成一张信号结果卡；结果卡支持逐字解码最后一段信号，也可以重置并重新开始。重置会清除旧结果和未完成状态，并把焦点送回第一颗信号节点。

## 布局与可访问性

- 使用原生按钮实现信号节点、解码和重置操作。
- 通过 `aria-pressed` 同步信号节点的选择状态。
- 使用简短的可见状态和独立 live region 传达进度、结果及解码完成信息。
- 解码期间保留按钮焦点，并阻止重复启动；重置后恢复明确焦点。
- 自动漂移动画只运行一次，并通过 `prefers-reduced-motion` 支持减少动态效果的系统偏好。
- 关键隐私文字使用固定、不透明的前景色和背景色，避免对比度随页面位置变化。
- 桌面和移动竖屏布局均支持完整的点亮、结果、解码和重置流程。

这些实现改善了当前页面的可访问性，但不代表项目已经获得完整 WCAG 认证。

## 文件结构

- `index.html`：页面结构、信号节点、结果卡和状态区域。
- `style.css`：夜间视觉、动画、焦点样式和响应式布局。
- `script.js`：节点状态、结果生成、解码、重置和异步状态管理。
- `README.md`：项目说明、运行方式和验证范围。

## 本地运行

直接使用浏览器打开 `index.html` 即可运行。项目没有构建步骤，也不需要安装依赖；也可以通过任意本地静态文件服务预览。

项目适合直接部署到 GitHub Pages 或其他静态站点服务。
